How to deploy (GitHub → Netlify):
1) Download this ZIP and unzip it.
2) In your repo `spre-platform-site`, click **Add file → Upload files**.
3) Drag these 4 paths into the repo root and commit to `main`.
4) After Netlify publishes, open: https://www.spre-platform.com/spre-dashboard.html

Update after each drill:
- Overwrite `/spre-cs-data/coldstart/latest.json` with the latest results.
